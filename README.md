# commute_map
This python script is used to generate a heatmap for time needed to commute from a given location.
The Google Maps API is used as the source of data, it requires a Google Maps API key.

For documentation and modification of this script, please refer to [Python Client for Google Maps Services](https://github.com/googlemaps/google-maps-services-python)

<h2>System requirements</h2>
  Python 3.5+
  
  Google Cloud Console
  
  Python package googlemaps
